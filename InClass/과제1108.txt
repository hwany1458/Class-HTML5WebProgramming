[실습과제 - 조건문/반복문] 

1. 임의의 배열을 선언하고, 좋아하는 과일 4개를 입력(초기 할당)하여 출력
2. (1번)에서 3번째 요소의 값을 “디지털콘텐츠”로 수정하고 배열 내의 모든 요소를 출력
3. 임의의 정수(0~100)를 입력 받아, 성적(A/B/C/D/F)을 출력
4. 입력 받은 정수가 짝수인지 홀수인지를 출력
5. 0부터 입력 받은 정수까지의 누적 합을 계산하여 출력
6. (5번) 누적 합을 구할 때, 짝수인 누적 합을 계산하여 출력


[실습과제 - 함수] 

1. 2개의 정수를 받아, 큰 값을 돌려주는 함수를 작성하고 이를 호출
2. 호출할 때, 정수 값을 적절하게 넘겨줍니다
3. 1개의 정수를 받아, 절대값을 돌려주는 함수를 작성하고 이를 호출
4. 1개의 정수를 받아, 값만큼 이름을 출력하고 (반환 값 없이) 함수가 종료되는 함수를 작성하고 이를 호출

5. 임의의 정수(2~9)를 입력 받아, 구구단을 출력
